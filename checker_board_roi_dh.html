checker_board_roi_dh.get_roi_filtered_pcd_with_checkerboard(pcd, transform_camera_to_checkerboard, roi_box_bottom_left_corner_x, roi_box_bottom_left_corner_y, roi_box_bottom_left_corner_z, roi_box_length, roi_box_width, roi_box_height)

Function that returns ROI filtered point cloud, original point cloud, transformation matrix of ROI box from the camera’s point cloud. Note that this function make use of function ‘roi_box_pcd’

Parameters:

  • pcd – Input point cloud given in camera frame

  • transform_camera_to_checkerboard – Transformation matrix of checkerboard coordinate system relative to camera frame

  • roi_box_bottom_left_corner_x – Distance in X coordinate from the board origin to the bottom left ROI box corner in checkerboard coordinate system

  • roi_box_bottom_left_corner_y – Distance in Y coordinate from the board origin to the bottom left ROI box corner in checkerboard coordinate system

  • roi_box_bottom_left_corner_z – Distance in Z coordinate from the board origin to the bottom left ROI box corner in checkerboard coordinate system

  • roi_box_length – ROI box dimension in X axis of the checkerboard coordinate system

  • roi_box_width – ROI box dimension in Y axis of the checkerboard coordinate system

  • roi_box_height – ROI box dimension in Z axis of the checkerboard coordinate system

Returns:

ROI filtered point cloud transform_camera_to_roi_box: Transformation matrix of ROI box relative to camera frame

Return type:

pcd_filtered

checker_board_roi_dh.roi_box_pcd(pcd, roi_box_bottom_left_corner_x, roi_box_bottom_left_corner_y, roi_box_bottom_left_corner_z, roi_box_dimension_in_axis_x, roi_box_dimension_in_axis_y, roi_box_dimension_in_axis_z)

Filter point cloud based on ROI box by providing box location and dimensions.

This function assumes that the point cloud is transformed to the checkerboard frame.

Parameters:

  • pcd – Input point cloud which is transformed to the checkerboard frame (Open3D format)

  • roi_box_bottom_left_corner_x – Distance in X coordinate from the board origin to the bottom left ROI box corner in checkerboard coordinate system

  • roi_box_bottom_left_corner_y – Distance in Y coordinate from the board origin to the bottom left ROI box corner in checkerboard coordinate system

  • roi_box_bottom_left_corner_z – Distance in Z coordinate from the board origin to the bottom left ROI box corner in checkerboard coordinate system

  • roi_box_dimension_in_axis_x – ROI box dimension in X axis of the checkerboard coordinate system

  • roi_box_dimension_in_axis_y – ROI box dimension in Y axis of the checkerboard coordinate system

  • roi_box_dimension_in_axis_z – ROI box dimension in Z axis of the checkerboard coordinate system

Returns:

A masked numpy array of X, Y and Z point cloud coordinates (Nx3 ndarray) masked_rgba: A masked masked RGB image (Nx3 ndarray)

Return type:

masked_xyz
