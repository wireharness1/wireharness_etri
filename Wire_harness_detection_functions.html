Wire_harness_detection_functions module

Wire_harness_detection_functions.Convert_to_poses(pcd_branch, cloud, capture_pose, grasp_pose)

This function transforms a point cloud branch using given robot poses for capturing and grasping. It calculates the center and end points of the profile and determines the vector between them, aligning the profile with the robot’s grasp direction.

Input:

pcd_branch: The point cloud representing the branch to be transformed. cloud: Additional point cloud data (purpose not detailed in the function). capture_pose: The pose matrix representing the robot’s capture position. grasp_pose: The pose matrix representing the robot’s grasp position.

Output:

pcd_branch: The transformed point cloud after applying the capture pose. profile_center: The centroid of the cropped point cloud representing the profile’s center. profile_center_to_end_vec: The normalized vector from the profile center to the profile end. profile_vector_len: The length of the vector from the profile center to the profile end. cloud: The additional point cloud data (passed through unchanged).

Wire_harness_detection_functions.Detect_D1C1_wire_onboard(pcd, visualize=0)

This function is for tracking the wire profile specified for D1C1 and now replaced by function Detect_wire_v2.

Wire_harness_detection_functions.Detect_D1C2_wire_onboard(ply_link, visualize=0)

This function is for 3D point clouds captured from D1C2 type, which have multiple branches and terminals existing in the input point clouds. After eliminating the background, it will apply color classification and then, split the parts of terminals and wires. Next, it will classify the terminals to know which colors to use for each branch. Finally, it will track each branch’s profile.

Input:

ply_link: The local directory of the input point cloud. visualize: If you want to see the detected profile =>, else => 0.

Output:

pcd_terminal_official: List of the point cloud of the terminals detected pcd_braches: List of the point cloud of the wire profile detected

Wire_harness_detection_functions.Detect_wire_v2(pcd, wire_type='D1C1', visualize=0)

This function is for tracking the wire profile on the inputed point cloud from the 3D camera. Firstly, the background will be eliminated based on multiple-plane detection. Then, the color classification model is applied to collect the points having colors corresponding to the target wire type. Finally, the wire profile is applied using the developed tracking method.

Input:

pcd: The input point cloud from 3D camera. wire_type: The target wire type. visualize: If you want to see the detected profile =>, else => 0.

Output:

pcd_branch: Profile of the wire point cloud cloud: Wire point cloud extracted

Wire_harness_detection_functions.ROI_center_extraction(pcd_profile, input_point, radius)

This function extracts the center of a Region of Interest (ROI) from a point cloud. It calculates the centroid of points within a specified radius around an input point.

Input:

pcd_profile: The input point cloud from which the ROI is extracted. input_point: The center point (x, y, z) around which the ROI is defined. radius: The radius defining the extent of the ROI around the input point.

Output:

[x_cen, y_cen, z_cen]: The centroid coordinates of the ROI within the specified radius.
