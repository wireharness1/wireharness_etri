classfify_terminal(input_surface_list, visualize=0)

This function takes the list of 3D point clouds from a 3D camera as inputs. Initially, it matches it with 3 template models. Finally, it outputs the class of each surface and the error of matching times.

Parameters:

  • input_surface_list (list[o3d.geometry.PointCloud]) – List of point clouds of the terminal from the 3D camera.

  • visualize (int) – If 1, displays the matching results; if 0, does not display. Defaults to 0.

Returns:

List of terminal classes for the inputted surface list and the error list from multiple matching results.

refine_registration(surface, model, surface_fpfh, model_fpfh, voxel_size, trans_init)

This function performs registration refinement between two point clouds using the ICP algorithm. The function refines the alignment of the ‘surface’ point cloud with the ‘model’ point cloud by iteratively minimizing the point-to-plane distance between them.

Parameters:

  • surface (o3d.geometry.PointCloud) – Point cloud of the surface scanned from the 3D camera.

  • model (o3d.geometry.PointCloud) – Point cloud of the model template.

  • surface_fpfh (o3d.registration.Feature) – Fast Point Feature Histograms of the surface point cloud (not used directly in this function).

  • model_fpfh (o3d.registration.Feature) – Fast Point Feature Histograms of the model point cloud (not used directly in this function).

  • voxel_size (float) – Voxel size used to downsample the point clouds (used here as the distance threshold).

  • trans_init (np.ndarray) – Initial transformation matrix to align the surface point cloud to the model point cloud.

Returns:

The result of the ICP registration containing the refined transformation matrix and fitness score.

Return type:

o3d.registration.RegistrationResult
